Nursing School Report

Chapter 5 Programming Literacy

Basic Summary: There is acknowldgement that having software literacy to import/transform data, accomplish analytic tasks, create plots, query databases, and create digital assets will facilitate publication and grant submissions. However, many faculty would like more resources to be more self-sufficient.

Some faculty have improved their knowledge via self-education or participating in the Big Data class though most want more experience and is frustrated with not being able to fully comprehend the code in available pipelines and/or how to easily branch out into ad-hoc analysis paths. Some have training in SAS or SPSS and remain attracted to the integreated nature of those tools but realize that R and Python are the more relevant frameworks for the Microbiome and Metabolomics work at hand. Several faculty have leveraged personal or prior work relationships to obtain programming and data cleaning assistance. In some respects this is viewed as being typical of the research excperience although having access to resources would be ideal. All of the junior faculty interviwed realize to generate the data necessary for an R-level grant application will involve learning more about coding.

There is an associated interest in attracting students who possess programming skills or are at least enthusiastic about a learning path that involves programming. This is both to faciliate research and to make the student experience more productive since analysis of experimental data rypes is likeley to be a major theme in most thesis projects. It is important to note that such students are not being seen as a substitute for more formal types of support.
